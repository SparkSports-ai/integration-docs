If your casino operates on **Upgaming**, you do not implement the direct integration APIs described in this guide.

## How it works

1. Request SparkSports activation in your Upgaming back office.
2. Upgaming calls SparkSports provider endpoints (`LaunchGame`, `GetGames`, etc.).
3. Wallet settlement flows through Upgaming's reverse API.
4. Players see SparkSports in the casino lobby and play via Upgaming's iframe routing.

## What you need to do

Contact your **Upgaming account manager** to enable SparkSports for your brand. Provide:

- Casino name and slug
- Supported currencies
- Desired stake limits and win caps
- Player ID format used on your platform

Upgaming coordinates credential provisioning with SparkSports. You do not communicate with SparkSports directly for API credentials.

## Technical reference

Upgaming-facing API specifications are maintained separately between Upgaming and SparkSports. If you are an Upgaming operator, your technical contact at Upgaming is the right entry point.

For direct API integration (without Upgaming), see [Integration models](/docs/integration-models).
